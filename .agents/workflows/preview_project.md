# Preview Project Workflow

This workflow details the steps required to start the development server and preview the Dental Balance project.

## Steps

1. **Start Development Server**
   - Run the development server command.
   ```bash
   npm run dev
   ```
   - The server will typically be available at `http://localhost:3000`.

2. **Verify Preview**
   - Open [http://localhost:3000](http://localhost:3000) in your browser.
   - Ensure the page renders correctly and the "Dental Balance" header is visible.
