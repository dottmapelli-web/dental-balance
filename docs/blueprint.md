# **App Name**: Dental Balance

## Core Features:

- Dashboard: Dashboard page with visual overview of financial status.
- Transactions: Transactions page for detailed income and expense tracking.
- Monthly Summary: Monthly Summary page with detailed report, and LLM tool that flags anomalies or inconsistencies.
- Annual Summary: Annual Summary page with historical overview for the "Studio De Vecchi & Mapelli".
- Budget & Objectives: Budget and Objectives page for setting financial targets.

## Style Guidelines:

- Primary color: Light, desaturated blue (#A7D1EA) to create a calming and trustworthy atmosphere.
- Background color: Very light blue (#F0F8FF) for a clean, professional look.
- Accent color: Navy blue (#000080) to give visual interest and indicate growth.
- Font pairing: 'Inter' (sans-serif) for body text and 'Literata' (serif) for headers, creating a balance between modernity and readability.
- Clean and well spaced layout.
- Simple and elegant icons for easy navigation.
